RabbitMax

RabbitMax provides open source hardware and software for Internet of Things. Explore our products, fork the source code in GitHub or just start scrolling and learn more.

About

RabbitMax is an innovative project for Internet of Things which integrates open source hardware and software. The source code is available in GitHub.

Raspberry Pi HAT

RabbitMax 1

Open source hardware HAT for Raspberry Pi with a relay, IR, buzzer, RGB LED, slots for 1602 LCD display and up to 3 I2C sensors.

Sensors

HTU21D

HTU21D is easy to use I2C sensor for measuring temperature and humidity.

BMP180

BMP180 is easy to use I2C sensor for measuring temperature and barometric pressure.

TSL2561

TSL2561 is an advanced digital light I2C sensor.
