RabbitMax

RabbitMax provides open source hardware and software for Internet of Things based on Raspberry Pi. Explore our products, fork the source code in GitHub or just start scrolling and learn more.

About

RabbitMax is an innovative project for Internet of Things which integrates open source hardware and software. All source code is available in GitHub.

RabbitMax Flex is a Raspberry Pi HAT with all must have ingredients for Internet of Things and slots for up to 5 plug and play sensors. It will soon be on sale through IndieGoGo campaign.

Raspberry Pi HAT

RabbitMax Flex

Flexible HAT for Raspberry Pi suitable for do it yourself (DIY) weather station, automated desk assistant or prototyping Internet of Things (IoT).

RabbitMax Flex has a relay, transmitting IR LED and receiving IR photo sensor, buzzer, button, RGB LED, slot for modular LCD character display and slots for up to 5 plug and play I2C sensors.

The software of RabbitMax Flex is fully compatible with Raspian GNU/Linux distribution. There are open source examples written in the C programming language for controlling all components of RabbitMax Flex. Furthermore there is a custom GNU/Linux distribution for IoT which is based on Poky and built with OpenEmbedded and the Yocto/Project. Open source software for measuring and reporting data from the sensors over the lightweight messaging protocol MQTT is also available.

The Story

My name is Leon Anavi and I am a software engineer. I am passionate about various open source technologies and you might know me from FOSDEM, OpenFest or other open source events. I also maintain a YouTube channel and unfashionable old style personal blog where I publish geek videos and tutorials for programming and do it yourself (DIY) devices. I have been a proud Raspberry Pi owner since 2011 and I enjoy working on hobby projects with it in my spare time.

RabbitMax started as a hobby IoT project almost a year ago, at the end of 2015. Initially I bought Raspberry Pi Sense HAT and I built a weather station with it. Unfortunatelly I noticed several limitations of Sense HAT for my needs. For example the data from the temperature sensor was not accurate because it is situated near the CPU of Raspberry Pi which heats over the time. The 8Ã—8 RGB LED matrix was also not very convenient for my needs. Soon I understood that Raspberry Pi Sense HAT is a great product but it is not exactly what I need for rapid IoT prototyping. There was only one way to move forward and I started working on my own Raspberry Pi HAT.

Making hardware is not an easy task, especially for a software engineer like me, so I asked my friends for help. As an open source enthusiast I insisted on using open source software for electronics design automation suite. Furthermore I wanted a software that runs on GNU/Linux distributions. Therefore RabbitMax Flex, my first Raspberry Pi HAT, is being designed with KiCAD.

The first prototype was with couple of sensors, relay, buzzer, RGB LED and simple LCD display attached to Raspberry Pi 2 through a breadboard and thousands of jumper wires. After that using scissors I created Raspberry Pi HAT cardboard sample. This dummy sample was very useful and helped me to select appropriate locations for the components.

The next step was to start a quest for a hardware engineer who really understands and appriciates the value of open source. Based on my experience this was very difficult. Furthermore RabbitMax is still just a hobby project so I have a very limited budget and it was even more difficult. In couple of months I met dozen of hardware engineers. Most of them were familiar with propriatery software tools such as EAGLE or Altium and they had absolutely no intention of using KiCAD instead. Finally, after asking around like-minded open source developers, I found the right person for my project and I convinced him to explore and use KiCAD for this project.

In March 2016 we started working on the PCB design of the add-on board for all Raspberry Pi models and versions with 40 pin headers. Simultaniously I was searching for an appropriate name of the project. Intially I called it RabbitPi but soon I figured out it is not that convenient and I renamed the whole project to RabbitMax. I also had to figure out a name for the first Raspberry Pi HAT model and I decided to call it RabbitMax Flex due to the ability easily to add or remote sensors and LCD displays.

RabbitMax Flex has a relay, transmitting IR LED and receiving IR photo sensor, buzzer, button, RGB LED, slot for modular LCD character display and slots for up to five addional I2C sensors. Following the guidelines of Raspberry Pi Foundations all add-on boards called HAT should have a EEPROM with some system information to identification.

The first real PCB prototype was printed in OSH Park. By the way I am verry happy with the service provided by OSH Park. The quality is excellent and the pricing is very affordable for prototypes and hobby projects. Unfortunatelly revision 1.0 of RabbitMax had a few hardware bugs. We decided to change the buzzer and the button for revision 1.1 as well as to add a potentiometer for adjusting the backlight of the LCD module. In August 2016, after almost six months of work on the PCB design and after printing several prototypes, RabbitMax Flex is near a stable version...
