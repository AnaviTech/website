RabbitMax provides open source hardware and software for Internet of Things based on Raspberry Pi. Explore our products, fork the source code in GitHub or just start scrolling and learn more.

About

RabbitMax is an innovative project for Internet of Things which integrates open source hardware and software. All source code is available in GitHub.

RabbitMax Flex is a Raspberry Pi HAT with all must have ingredients for Internet of Things and slots for up to 5 plug and play sensors. It will soon be on sale through IndieGoGo campaign.

Raspberry Pi HAT

RabbitMax Flex

Flexible HAT for Raspberry Pi suitable for do it yourself (DIY) weather station, automated desk assistant or prototyping Internet of Things (IoT).

RabbitMax Flex has a relay, transmitting IR LED and receiving IR photo sensor, buzzer, button, RGB LED, slot for modular LCD character display and slots for up to 5 plug and play I2C sensors.

The software of RabbitMax Flex is fully compatible with Raspian GNU/Linux distribution. There are open source examples written in the C programming language for controlling all components of RabbitMax Flex. Furthermore there is a custom GNU/Linux distribution for IoT which is based on Poky and built with OpenEmbedded and the Yocto/Project. Open source software for measuring and reporting data from the sensors over the lightweight messaging protocol MQTT is also available.
